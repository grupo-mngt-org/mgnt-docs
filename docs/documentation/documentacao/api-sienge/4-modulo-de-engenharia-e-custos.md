# 4. Módulo de Engenharia e Custos

A inteligência da construção.

- **Orçamentos:** Planilhas orçamentárias (itens, composições e insumos).
- **Acompanhamento/Medições:** Medição de contratos de serviços e evolução de obra.
- **Cronogramas:** Datas e etapas planejadas vs. realizadas.
