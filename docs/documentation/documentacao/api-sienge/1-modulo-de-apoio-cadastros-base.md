# 1. Módulo de Apoio (Cadastros Base)

Estes são os dados que alimentam todos os outros módulos. Sem eles, as outras APIs não funcionam.

- **Empresas e Unidades Organizacionais:** Cadastro de holdings, empresas e filiais.
- **Projetos e Obras:** Estrutura de centros de custo e locais de obra.
- **Pessoas:** Fornecedores, clientes, funcionários e contatos.
- **Produtos e Serviços:** Catálogo de insumos e serviços prestados/contratados.
