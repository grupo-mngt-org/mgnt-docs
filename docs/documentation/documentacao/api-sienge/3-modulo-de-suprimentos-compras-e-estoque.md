# 3. Módulo de Suprimentos (Compras e Estoque)

Focado na cadeia de suprimentos da obra.

- **Requisições:** Solicitação de materiais.
- **Cotações:** Processo de escolha de fornecedores.
- **Pedidos de Compra:** Fechamento de pedidos e contratos.
- **Notas Fiscais:** Recebimento físico e fiscal (entrada de mercadorias).
- **Estoque:** Movimentação entre almoxarifados.
