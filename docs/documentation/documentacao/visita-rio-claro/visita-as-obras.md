# VISITA AS OBRAS

- **Acesso as obras**
    - Controle de acesso, temos o Mobuss usando o mobbus catraca
    - Gabriel tem outra aplicação para fazer isso
- **Armazenamento de versões de projetos**
    - Os projetos são atualizados no google drive, não ocorre aviso dessa atualização de quem está na obra e eles ficam utilizando o projeto antigo
    - Eles usavam o AUTODOC
- **Requisição de retirada de insumo**
    - A requisição de compra é feito via papel
    - Registro dessas requisições
    - Implementação de saída de insumos a partir de qrcode
- **Controle de notas fiscais para estoque (Parece que planilha atende)**
    - Existe um problema quando o fornecedor emiti a nota fiscal e tem que acontecer o faturamento da mesma, os casos são os seguintes:
        - Nota emitida pelo fornecedor com grande antecedência da entrega, é necessário que tenhamos esse controle para poder pedir o cancelamento dessa nota
        - Nota emitida pelo fornecedor, recebimento na obra ainda não ocorreu e financeiro fatura a nota
        - Double check do engenheiro ser mais automatizado para tirar a obrigatoriaedade semanal de ir ao escritório levar as notas assinadas
