## O que é o n8n
É uma ferramenta de automação low-code, que permite conectar com aplicações, bancos de dados, API's e etc. Com uma gama enorme de soluções prontas para integração, e em especial se integra bem com ferramentas de LLM.
Pode ser usado self hosted ou em cloud.

## Uso do n8n no Grupo MNGT
O n8n deve ser usado para automatizações que não exigem complexidade avançada, integrando com ferramentas utilizadas no Grupo MNGT, como: Discord, Whatsapp, Google Sheets, Google Calendar, Email, APIs e etc...

## Aplicação para Inteligência Artificial
O interesse do Grupo MNGT com o uso do n8n, além de ser uma ferramenta que agiliza e simplifica automatizações, é principalmente a fácil integração que o n8n tem com ferramentas de LLM como o OpenAI. Dessa forma, é possível utilizar tanto para integrar nodes com LLM quanto para criar de fato agentes de IA.

## Acesso
O Grupo MNGT utilizar o n8n self hosted na versão community. A orientação é que todos os desenvolvedores utilizem a conta padrão para o Grupo, que é o administrador da instância, que pode ter as credenciais encontradas nas docs.

Com a conta de admin é possível gerenciar os workflows e as credenciais, portanto é preciso cuidado ao utilizar, com o entendimento que é uma conta de uso coletivo.
